## ✨ Features

- 🌙 Dark / Light mode toggle
- 🤖 Real Bitmoji avatar with animated floating badges
- ⌨️ Animated typing effect cycling 4 titles
- 🎯 3 featured project cards — Problem → Architecture → Achievements
- 📊 Animated engineering metrics counters
- 🗓️ Learning journey timeline
- 📈 Live GitHub stats integration
- 📬 Contact form with direct links
- 📱 Fully mobile responsive
- ⚡ Framer Motion scroll animations
